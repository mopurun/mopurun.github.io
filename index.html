👋 自己紹介

こんにちは、もぷるん（mopurun）です。
元・電子系エンジニアで、現在は趣味で電子工作とプログラミング、電子部品の収集、お絵かきなどをしています。低レイヤの技術やマイコンが大好きです。

🔧 関心のある分野

  • アセンブリ言語（PC・マイコン両方）
  • 電子工作（基板設計、マイコン制御など）

🧪 少し興味がある技術

  • プログラミング言語：C / C++ / C# / Python / JavaScript / Rust / VBA / bat /　sh / WebAssembly
  • ハードウェア記述言語：VHDL / SystemVerilog
  • Web技術：HTML / CSS / SSG(Hugo etc...)
  • 情報セキュリティ：アセンブリ言語（Pwnable、マルウェア解析）やハードウェア関連とか面白そうだな～と思う
